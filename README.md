## Process

We are going to link this repository with the above mentioned using a `git submodule`. For that we will need to run the following command in each of the required repositories

```bash
git submodule add https://github.com/alcb1310/proto-def.git
```

This command will add the contents of this repo with its most recent commit to the project.
